# Amazon IAM Identity Center (amazon-iam-identity-center)
AWS IAM Identity Center (successor to AWS Single Sign-On) is where you create, or connect, your workforce identities in AWS once and manage access centrally across your AWS organization. You can create user identities directly in IAM Identity Center, or you can bring them from your Microsoft Active Directory.

### AWS IAM Identity Center API
The AWS IAM Identity Center API provides programmatic access to manage permission sets, assignments, instances, accounts, and SSO configurations for centralized identity and access management.
